# webcn-i-lab

本仓库用于归档和发布多个独立的 HTML 页面。这些页面可能包含实验性功能、样式测试、工具演示或其他前端相关内容的静态实现。

## 项目简介

webcn-i-lab 是一个轻量级的 HTML 页面集合仓库。每个页面均可独立运行，不依赖外部后端服务。仓库以文件归档为主要目的，便于对各类前端页面进行版本管理与历史回溯。

## 目录结构

```
├── pages/          # 存放各个独立 HTML 页面及其相关资源
├── README.md       # 本文件
└── LICENSE         # 开源许可证
```

- `pages/` 目录下每个子文件夹代表一个独立页面，包含对应的 HTML、CSS、JavaScript 及静态资源。
- 页面命名与组织方式以清晰可辨为原则，不强制统一格式。

## 页面归档说明

- 每个页面均为完整的、可离线运行的 HTML 文件（或包含必需资源的文件夹）。
- 页面内容可能处于不同完成阶段：包括实验原型、功能演示、样式探索等。
- 归档时不承诺所有页面均具备生产环境可用性，亦不保证长期兼容性。

## 如何浏览

你可以通过以下方式查看已归档的页面：

1. **直接克隆仓库**，在本地浏览器中打开 `pages/` 下的 HTML 文件。
2. **启用 GitHub Pages**（如已配置），通过仓库设置中的 Pages 功能访问在线版本。
